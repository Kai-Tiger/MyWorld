# 雪山山脚平缓化

## 背景与目标

参考图的山从草原平缓抬升;原版 4 座雪峰朝玩家(南)的正面是近垂直的"竖条幕布墙"。目标:在保住峰高前提下，把山面陡度降到 ≤~60°、消除竖墙。

## 已确认的几何事实（多轮实测）

- 山面陡度 = `峰高 ÷ 峰到山脚水平距离`。雪峰 ~190–315m。
- 竖墙主因之一是高度修改器里的**切底 `-100`**：把穹顶下半段缓坡削掉，山从 100m 等高线突兀冒出。
- massif（4 峰 + 5 山脊 + 英雄河源头锚定在峰 + 多层河流下切）高度耦合：
  - 移峰/扩图/整体北移 → 簇峰 `Math.max` 并集成"高原+连续陡边墙"，更糟（实测）。
  - 加宽宽谷 → 抬高山脚基底、峰顶整体升高（实测，反向）。

## 已交付改动（低风险，零河流/世界改动）

`src/scene/map.js`：
- `applySnowMountainHeight` 切底 `mountainLift - 100` → `- 35`：保留穹顶下半段缓坡，山脚不再从 100m 等高线突兀冒出。
- `SNOW_PEAKS` 各峰 `h` 同步下调 65 补偿，使净峰顶高度不变（`base+h-100 == base+(h-65)-35`）：250/190/167/123（原 315/255/232/188）。

**效果**：非河谷侧（左侧 peak1/peak3）山脚明显变缓；峰心高度保持（实测 337/258/280/243 ≈ 原 341/259/282/243，±4）。同样高度摊在更大穹顶半径 → 山面更缓。

## 已知未解决（需后续专门处理）

- **东南沿英雄河峡谷一侧仍为陡墙**：这是河流下切的峡谷壁，非山体穹顶，切底软化覆盖不到。
- 软化峡谷壁需重做已提交、精调过的多层河流下切（`applyLargeRiverValleyHeight` / `applyHeroRiverHeight` / 河道/冲沟碳刻），耦合极深：本次以参数迭代未能可靠收敛（加宽宽谷会抬高峰、引入新台阶）。建议作为**独立的、可对每个修改器逐项插桩**的专门任务处理；或采用降峰高(路径 B)直接获得参考图式缓坡。
