# 只使用森林包中最像草的资源铺出生点

## 背景

森林资源包没有独立草贴图，只有 GLB 植被模型。用户希望尽量排除灌木，只保留像草的资源铺在出生点附近。

## 目标

- 只使用森林包中最接近草的 `background_tree_12.glb`。
- 在出生点 `(0, 40)` 周围 `10m` 内生成低矮草丛。
- 避开玩家初始脚下中心区域。
- 不添加碰撞，不影响玩家移动。

## 实施

- 在 `src/scene/map.js` 新增出生点草丛配置和生成函数。
- 使用确定性随机分布生成 34 个低矮草丛实例。
- 每个实例随机旋转和缩放，缩放范围控制在 `0.18 - 0.32`。
- 使用 `cloneGLTFScene` 加载森林包 GLB，并用 `snapObjectToGround` 贴合地形。

## 验收

- `npm run build` 成功。
- 出生点附近出现低矮草丛。
- 不出现明显灌木、小树或高植被。
- 玩家移动不被草阻挡。
- 浏览器控制台没有 GLB 加载错误。

## 假设

- “只要草”优先解释为只使用森林包里最接近草的 `background_tree_12.glb`。
