# 拾取动画与拾取反馈

## 背景

拾取物品时原逻辑会立即移除光球，靠近提示也直接展示具体物品名。新增 `player/pick.fbx` 后，需要让拾取动作、光球淡出和获得提示形成完整反馈。

## 目标

- 按 `H` 或点击提示拾取时播放 `player/pick.fbx`。
- 拾取提示只显示 `捡起`。
- 拾取成功后光球淡出，不瞬间消失。
- 拾取成功后在屏幕中下方显示物品名称，1.5 秒后消失。
- 拾取动画期间禁止移动、攻击、翻滚、施法、治疗。

## 实施

- 将 `pick.fbx` 接入玩家动画系统，并加入运行时预加载清单。
- 新增 `player.playPick()` 和 `player.isPicking()`，按一次性动作播放。
- 拾取触发时先播放拾取动画，成功后标记拾取物已拾取、入背包、展示物品名称。
- 扩展拾取光球特效，支持 `fadeOut(duration)`，淡出结束后释放资源。
- 在 UI 层新增拾取成功提示，显示物品名称 1.5 秒。

## 验收

- 靠近光球时按钮文案为 `捡起`。
- 按 `H` 后玩家播放拾取动画。
- 光球渐隐后消失。
- 屏幕中下方显示 `新法术卷轴`，约 1.5 秒后消失。
- 拾取后背包内出现对应物品，且不能重复拾取同一光球。
