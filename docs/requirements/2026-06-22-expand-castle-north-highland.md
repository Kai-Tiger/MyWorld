# 扩大城堡前北侧高地

## Context and Goal

城堡前北侧高地面积偏小，需要扩大到约原来的两倍，同时不能挡住矿洞入口、梯子交互或洞口周边清理区。

## Success Criteria

- 高地顶部和可走区域明显扩大，面积约为原来的两倍。
- 矿洞入口附近不被高地地形抬升覆盖。
- 洞口附近不新增树木、岩石或碰撞物。
- 高地上的敌人仍在高地区域内，且敌人数量不增加。

## Planned Changes

- 扩大 `CASTLE_NORTH_HIGHLAND` 的椭圆半径，并将中心略微向东/北移动。
- 在高地高度贡献上加入矿洞清理圈衰减，让洞口附近保持低地。
- 为扩大后的高地补充少量树木和岩石摆件。
- 微调北侧高地敌人/巡逻点到扩大后的平台范围内。

## Test/Verification Plan

- 静态检查高地参数、洞口避让半径和新增摆件坐标。
- 运行 `npm run build`。

## Assumptions and Out of Scope

- “扩大一倍”按面积约翻倍处理，不按半径翻倍处理。
- 不修改矿洞模型、矿洞碰撞、城堡入口、主路线或敌人总数。
