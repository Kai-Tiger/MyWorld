# 城堡门前白雾门流动朦胧效果

## 背景

城堡入口白雾门已有透明 shader 和多层 plane，但视觉偏平，缺少雾团层次和持续流动质感。

## 目标

- 白雾门正面能看到持续流动的雾团变化。
- 近距离观察不再像单层白色半透明板。
- 侧面观察没有明显穿帮、黑边或异常闪烁。
- 不改变进入城堡的交互、碰撞和传送逻辑。

## 实施

- 增强现有雾门 shader，使用多层 procedural noise、domain warp、横向漂移和纵向流动生成朦胧雾团。
- 为 4 个雾门层设置轻微前后错位、不同透明度、速度、相位和尺寸，形成深度叠层。
- 在雾门更新逻辑中增加轻微层间漂移和呼吸缩放，强化流动感。
- 保持现有透明渲染策略，不新增贴图资产。

## 验收

- `npm run build` 成功。
- 城堡门前白雾门有明显但克制的动态雾团和朦胧层次。
- 玩家靠近雾门后原有进入城堡逻辑仍正常。
- 浏览器控制台没有新增 shader 编译错误或运行时错误。

## 假设

- 本次只增强白雾门视觉质感，不改成火焰、黑雾或实体门。
- 使用 shader 自生成纹理，避免新增资源加载和资产管理成本。
