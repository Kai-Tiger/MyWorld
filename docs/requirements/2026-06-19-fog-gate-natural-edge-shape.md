# 白雾门自然弧形边缘

## 背景

白雾门已使用单层 `fog.png` 贴图 shader，但边缘仍按矩形 plane 的 UV 边界衰减，视觉上像一个长方形。

## 目标

- 白雾门不再出现明显长方形边界。
- 顶部和两侧有自然弧度与轻微起伏。
- 中心区域仍保持浓雾，能遮挡门后内容。
- 不改变雾门 mesh 尺寸、位置、交互、碰撞或传送逻辑。

## 实施

- 用 shader 内的拱形轮廓遮罩替代直线矩形边缘衰减。
- 使用低成本动态三角函数给顶部、侧边和底部边缘增加轻微起伏。
- 保留现有 `fog.png` 双 UV 流动采样和浓雾 alpha 参数。

## 验收

- `npm run build` 成功。
- 城堡门前白雾门边缘有自然弧度，不再像硬长方形。
- 雾门表面仍有流动雾纹，中心仍足够浓。
- 浏览器控制台没有新增 shader/WebGL 编译错误。

## 假设

- 本次只修改视觉轮廓，不改变实际门洞范围或物理逻辑。
