# 空中脚步骤抑制

## 背景与目标

跑步脚步声音频可以继续同时玩家是跳跃。 脚步骤循环应停止从跳跃启动到落地，然后恢复正常之后玩家落地了。

##成功标准

- 跑步和行走循环同时停止玩家是空中的。
- 按住移动在空中不重启脚步声循环。
- 脚步骤循环恢复落地后移动状态允许它们。
- 现有非跳跃脚步声抑制行为是固定。

## 计划

- 新增一个玩家`isAirborne()`查询该覆盖不地面的，保持实现跳跃动画，和跳跃起飞发条。
- 包含`player.isAirborne?.()`在`updatePlayerFootsteps()`停止条件。

## 测试与验证计划

- 运行 `npm run build`。
- 在游戏中，运行，跳跃，hold 移动在空中，和确认运行音频停止 until 落地。

## 假设与范围外

- Landing 帧可以恢复脚步声立即如果玩家是仍然移动。
- 无落地音效或跳跃音效改动是包含。
