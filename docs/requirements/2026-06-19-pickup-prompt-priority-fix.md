# 拾取提示优先级修复

## 背景

拾取光球位于出生点附近，并且靠近 NPC。原逻辑把 NPC 对话范围计入拾取提示的阻断条件，导致玩家越靠近光球旁边的 NPC，拾取提示反而被隐藏。

## 目标

- 玩家进入拾取范围后稳定显示拾取提示。
- 拾取提示优先于 NPC 对话提示。
- 门、城堡入口、篝火等关键交互仍然优先于拾取提示。
- 按 `H` 拾取后，光球消失并进入背包。

## 实施

- 在室外主循环中先计算当前可拾取物。
- `updatePickupInteraction` 接收已计算的拾取目标，避免重复查找和不清晰的阻断逻辑。
- NPC 不再阻断拾取提示；当拾取提示存在时隐藏 NPC 对话按钮。
- 城堡入口、门、篝火仍作为拾取提示阻断条件。

## 验收

- 靠近 `(1.8, 36)` 的光球时显示 `拾取：新法术卷轴 [H]`。
- 贴近光球旁边 NPC 时，拾取提示不消失。
- 拾取后光球消失，法术卷轴进入背包。
- 拾取后仍可正常与 NPC 对话。
