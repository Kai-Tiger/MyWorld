# 远处辫状地形裁切修复

## 背景与目标

远处敌人区域附近 `(-448.9, -474.9)` s其内部参考地形雕刻 region。 运行时 probing 显示的本地地形高度改动由向上到约 `1.26m` 上方一个 `1m` 采样步骤，同时玩家 auto-步骤阈值是`0.55m`。 视觉地形网格 interpolates 这些变窄裁切，但玩家地面采样 analytic 高度，causing wave-类似地面和 apparent 裁切。

目标: 柔和en high-frequency 参考地形裁切在这播放able pocket 足够用于地形网格和玩家地面到 agree，且不改变玩家步骤行为或移动河流路径。

## 成功标准

- 地面周围 `(-448.9, -474.9)` 不再显示变窄 wave-类似裁切该玩家可以明显 sink 进入。
- 玩家可以行走围绕远处敌人区域不带明显地形裁切 ca使用d 由辫状河流裁切。
- 附近水和地形系统保持连续，带有无新明显干洞或悬空水水片。

## 计划改动

- 新增一个小本地 attenuation 遮罩围绕远处敌人区域用于参考雕刻's high-frequency `fineCut`。
- 保持现有河流路径，深度，和水渲染 intact。
- 不要改变 `AUTO_STEP`; 这是一个地形-塑形问题，不一个玩家移动 permission 问题。

## 测试与验证计划

- Sample 地形附近 `(-448.9, -474.9)` 之前和之后改变和对比本地高度跳跃。
- 运行项目构建到 catch syntax/module regressions。
- 如果实际可用，检查远处敌人区域在浏览器之后启动。

## 假设与范围外

- 报告的截图是从远处敌人区域附近 `(-448.9, -474.9)`。
- 草密度和草放置是不 root ca使用和不在范围内。
- 宽阔重新设计的新区河流，湖，or 玩家移动不在范围内。
