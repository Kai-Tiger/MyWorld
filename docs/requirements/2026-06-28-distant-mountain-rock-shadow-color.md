# 远景山体岩石阴影颜色

## 背景与目标

远景覆雪群山读作过于平坦和白色从长视角。 它们应 retain 一个雪顶 identity 同时显示更多灰色岩石，冷色阴影，和山脊/冲沟变化。

目标: improve 远景地形代理着色器因此远处群山有更强岩石和阴影颜色不带新增新几何体或改动地形高度。

## 成功标准

- 远景群山是不再大多 pure 白色从报告的视角。
- 陡峭坡面和破碎雪条带显示更多冷色灰色岩石。
- 雪阴影是明显冷色er 和更深色，带有山脊/冲沟变化。
- 雪 caps 保持recognizable 在高山峰。
- 附近森林，河流，天空，和地形几何体保持不变。

## 计划改动

- 调整 `createDistantTerrainProxyMaterial()` 雪和岩石颜色 mix。
- 降低雪覆盖在陡峭和破碎 high-altitude 表面。
- 提高代理雪 relief contrast 和冷色 blue-灰色阴影。
- 更新远景地形代理着色器缓存关键。

## 测试与验证计划

- 运行 `npm run build`。
- 加载游戏和检查报告的长山体视角。
- 确认浏览器控制台有无着色器编译错误。
- 确认前景地形，森林，天空，和水是不视觉上或可用ly 改变。

## 假设与范围外

- 这是仅限着色器的视觉微调。
- 无 entity 放置，地形高度，树木分布，河流，or 全局光照改动是包含。
- 无新岩石网格是新增到远景群山。
