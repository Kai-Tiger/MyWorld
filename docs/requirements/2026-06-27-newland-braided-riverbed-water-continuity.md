# 新区辫状河床与水连续性

## 背景与目标

新区辫状河流可以显示大型 polygonal 深色阻挡，锯齿状水孔洞，和不平整水下河床形状附近 `(-413.1, -326.8)` 和 `(-213.4, -386.8)`。 这区域是控制led 由统一新区辫状水网格。 Adjacent 网格 vertices 可以 choose 差异erent dominant 河槽，和河床可以 anchor 到本地高沙地而不是河槽中心，其中 makes 可见水 get 裁切进入矩形或triangular 水片。

目标: 使新区辫状水面和河床连续横跨附近/重叠河槽同时保留暴露沙地 bars 和现有河流路径。

## 成功标准

- `(-413.1, -326.8)` 不再显示大型步骤ped 深色水阻挡或锯齿状 polygon 孔洞。
- `(-213.4, -386.8)` 不再显示平坦沙地斑块 slicing 河流表面进入格挡y 水水片。
- `(-231.9, -477.7)` 保持一个柔和岸线不带矩形水水片。
- 新区辫状汇流区保持连接，但暴露沙地 bars 是不 flooded 进入一个宽阔湖。
- 河床下方可见水保持下方水面，因此平坦沙地斑块不再裁切水进入 rectangles或三角面。
- 主河流，分支，湖，山体 outlet，和玩法水 API 保持compatible。

## 计划改动

- 新增内部辅助函数用于新区辫状 candidate collection 和 weighted 水体-表面混合。
- 使用混合的新区水面在`buildNewlandBraidedWaterGeometry()` 而不是一个单层 best-河道中心采样。
- Drive 新区辫状河床雕刻从一个连续水体-表面 anchor: 核心河床 gets 稳定深度，浅水边缘保持just 下方水，和外侧河岸混合返回到地形。
- Anchor 每个 braided-河道水/河床目标从该河槽's pre-雕刻 centerline 高度，不当前点's 本地沙地高度。
- 清理模型草从新区辫状湿润河床因此草不 grow 通过河流河槽和 exaggerate 水/河床接缝。
- 保持现有统一水材质和合并水网格。

## 测试与验证计划

- 运行 `npm run build`。
- 检查 `(-413.1, -326.8)` 从报告的相机 angle。
- 检查 `(-231.9, -477.7)` 用于岸线 regression。
- 检查代表性辫状 junctions 周围 `(-390,-360)`，`(-300,-410)`，和 `(-220,-455)`。

## 假设与范围外

- 这可能调整新区辫状河床高度 locally，但不移动河槽控制点。
- 这不引入 `Water.js`，深度贴图，or 一个新水渲染系统。
- 这不重新设计旧 main-河流汇流池或静态湖。
