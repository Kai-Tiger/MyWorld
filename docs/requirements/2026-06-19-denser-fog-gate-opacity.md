# 提高白雾门视觉遮挡浓度

## 背景

当前白雾门已改为单层 `fog.png` 贴图 shader，但透明度偏高，门后场景仍容易透出来。

## 目标

- 白雾门中心区域视觉更浓，基本看不到门后内容。
- 保留 `fog.png` 的雾纹流动层次。
- 保留边缘羽化，避免出现硬矩形边框。
- 不改变进入城堡的交互、碰撞和传送逻辑。

## 实施

- 降低雾浓度阈值，让贴图中较暗区域也参与形成浓雾。
- 提高基础 alpha 和浓雾 alpha 权重，并限制最大透明度为 `0.98`。
- 略微提亮雾色底色，降低背后暗色物体透出的视觉对比。

## 验收

- `npm run build` 成功。
- 城堡门前白雾门明显更浓，正面基本看不到背后内容。
- 雾门仍有流动纹理，不是纯白静态板。
- 浏览器控制台没有新增 shader/WebGL 编译错误。

## 假设

- 本次只做视觉遮挡增强，不改变物理阻挡、相机遮挡或交互范围。
